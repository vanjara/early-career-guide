# Job Search Hub
### A practical guide for tech and business/finance new grads — including OPT/visa candidates

---

Most job search advice is either too vague to act on or buried in LinkedIn noise.

This is a structured, opinionated resource built for:
- Fresh graduates in **tech** (SWE, data engineering, data science, data analytics)
- Fresh graduates in **business and finance** (analysts, consulting, corporate finance)
- Candidates on **F-1 OPT or student visa** navigating job search with visa constraints

No hustle content. No paid courses. Just a practical system — what to do, in what order.

## The honest timeline

```
Actively searching, doing everything right:    8–12 weeks to an offer
Passively applying, Easy Apply only:           5–6 months, maybe longer
OPT candidates:                                Start before graduation
                                               Your clock starts at graduation day
```

If you're 3 weeks in and discouraged — that's normal.
If you're 3 months in with no traction — something in your system needs to change, not just your effort level.
